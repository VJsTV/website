# 3. Design Direction

* **Dark / cyber aesthetic**: black + neon purple, electric blue, magenta
* **Typography**: Inter, Space Grotesk, Orbitron
* **Visuals**: Video backgrounds, animated transitions, glitch effects, hover previews
* **Responsive**: Mobile-first, lazy-load videos/images

---

# 4. Submission / Community

* `/submit` page with Typeform/Tally integration.
* Fields: Artist, Project, Technology, Studio, Location, Video link.
* Submissions auto-generate entries in the GitHub repo via GitHub Actions.

# 5. Live Broadcast Page

* Dedicated `/live` page
* Embed YouTube/Vimeo stream
* List performing artists with links
* Display schedule of upcoming live events
