## Overview

**VJsTV** is the global platform for live visual performance, VJ culture, and audiovisual art. The website serves as a hub to:

- Stream **live VJ sessions** and curated audiovisual performances.  
- Explore **festival visuals** and live event coverage.  
- Connect **visual artists** with global audiences.  
- Provide a searchable **database of VJs, studios, and AV technologies**.  

Our mission is to become the **IMDb of Visual Performance**, centralizing the world of VJ culture, festival appearances, and live audiovisual innovation.

---

## Features

- **Live Streaming**  
  Watch curated VJ sessions and live performances in real-time.

- **Events & Festivals**  
  Browse upcoming and past events, including electronic music festivals, projection mapping shows, and immersive installations.

- **Artist Profiles**  
  Discover VJs, studios, and AV collectives worldwide with detailed profiles.

- **Tech Arsenal Database**  
  Explore the hardware, software, and AI tools used by visual artists globally.

- **Community Submissions**  
  Artists can submit visuals, projects, and festival performances for curation.

---

## Tech Stack

- **Frontend:** React, Next.js, Tailwind CSS  
- **Backend:** Node.js, Express  
- **Database:** MongoDB / Firebase (for live events and artist info)  
- **Streaming:** HLS / WebRTC integration for live visual sessions  
- **Deployment:** Vercel / Netlify  
- **Analytics & Monitoring:** Google Analytics, Sentry
